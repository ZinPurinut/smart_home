เนื้อหาคอร์ส 10 บท

01

บ้านอัจฉริยะคืออะไร

Foundation

ความหมาย ประโยชน์ ตัวอย่างการใช้งานจริง และเทรนด์ตลาด Smart Home ในไทย

02

ระบบสื่อสาร

Protocol

WiFi, Zigbee, Z-Wave, Bluetooth, Matter — ความแตกต่าง ข้อดีข้อเสีย และการเลือกใช้

03

สวิตช์ไฟอัจฉริยะ

Core

ประเภท การติดตั้ง การ Config บน Tuya App การตั้ง Scene และ Automation

04

ระบบแอร์

Core

IR Controller, Smart Thermostat, การเชื่อมต่อ, การตั้ง Schedule ประหยัดพลังงาน

05

ระบบม่าน

Core

Curtain Motor ประเภทต่างๆ การ Calibration, Zigbee vs WiFi Motor, การตั้ง Automation

06

ระบบรักษาความปลอดภัย

Security

Smart Lock, Video Door Phone, Auto Gate, Motion Sensor, การตั้ง Notification Alert

07

เซ็นเซอร์

Sensor

PIR, Door/Window, Smoke, Water Leak, Temperature — การติดตั้งและการตั้ง Automation

08

Gateway และ Home Assistant

Advanced

Zigbee Hub, Home Assistant OS, การ Integrate Tuya กับ HA, การใช้ Dashboard

09

การออกแบบระบบ

Design

การสำรวจบ้าน, การเขียน Proposal, การทำ Wiring Diagram, การประเมินราคา

10

Workshop

Practice

ฝึกปฏิบัติจริง Config ระบบตัวอย่าง ทดสอบ Scene, ทดสอบ Automation, แก้ปัญหาเบื้องต้น
